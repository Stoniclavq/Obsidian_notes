print - вывод в консоль текста или содержимое переменных, файлов
```python
print('Hello world')

w = 'world'

print(f'Hello {w}')

h = 'Hello'

print(h,w)
```

type - показывает тип переменной
```python
type(1)    # Вывод: int
type('hello')     # Вывод: str
```

input - команда для чтения входных данных с клавиатуры в виде строчных данных
 ```python
 my_data = input()
 
 name = input("Как вас зовут? ")    # в консоль выведется текст из скобок,  после чего мы также должны написать данные с клавиатуры
 
 
 ```
int(), float(), str() и т.д. - преобразовывает тип данных в тот, который мы укажем
```python
number = '72'
int(number)   # строковое number преобразаются в целочисленный формат

```
