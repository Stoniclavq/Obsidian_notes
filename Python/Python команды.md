print - вывод в консоль текста или содержимое переменных, файлов
```python
print('Hello world')
--------------
w = 'world'

print(f'Hello {w}') # выражение в этих скобках {} называется местозаполнителем

h = 'Hello'

print(h,w)
--------------
{местозаполнитель:спецификатор}
money_per_year = 5000
money_per_month = money_per_year / 12.0
print(f'Ежемесячный платяж: {money_per_month:.2f}') # Округляем число до 2 знаков после запятой с помощью спецификатора
# Вывод: Ежемесячный платяж: 416.67
--------------
number = 2233434324324
print(f'Число {number:,}')  # вставляет запятые в число для удобства чтения
# Вывод: Число 22,334,324,324
# При использовании спецификатора запятой и округления, сначала указывается запятая, а после спецификатор округления
--------------
number = 0.5
print(f'Число {number:%}') # умножает значения переменной на 100 и выводит со занком % в конце
# Вывод: Число 50.000000%
# Чтобы округлить до 0 знаков, вместо % пишем .0%
--------------
number = 2323232
print(f'Число {number:e}') # преобразовывает значение переменной к формату с e
# Вывод: Число 2.323232e+06
print(f'Число {number:.2e}') # преобразывает значение переменной к формату с e с двумя знаками после запятой
# Вывод:  Число 2.32e+06
--------------
number = 2323232  
print(f'Число {number:d}') # указаывает, что число целочисленное, еслт на вход поступает нецелочисленное число, то выводится ошибка
--------------
number = 100
print(f'Число {number:5}') # делает отступ слева от  числа на кол-во указанных символов
#Вывод: Число   100
--------------
# end - добавляет в конец вывода указанные элементы(по умолчанию перенос строки)
print('Один',end=' ')    
print('Два',end=' ')
print('Три')
# Вывод: Один Два Три
--------------
# sep - добавляет между элементами указанные элементы(по умол. пробел)
print('Один','Два','Три',sep='')
# Вывод: ОдинДваТри
```

type - показывает тип переменной
```python
type(1)    # Вывод: int
type('hello')     # Вывод: str
```

input - команда для чтения входных данных с клавиатуры в виде строчных данных
 ```python
 my_data = input()
 
 name = input("Как вас зовут? ")    # в консоль выведется текст из скобок,  после чего мы также должны написать данные с клавиатуры
 
 
 ```
int(), float(), str() и т.д. - преобразовывает тип данных в тот, который мы укажем
```python
number = '72'
int(number)   # строковое number преобразаются в целочисленный формат
 
```
