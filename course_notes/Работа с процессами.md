PID - Proccess ID
Ctrl+Z - свернуть процесс и поставить его на паузу
Ctrl+C - моментально закончить процесс
## В htop

\ (имя искомого процесса) - поиск по названию процесса
k - показывает все возможные сигналы, которые мы можем отправить процессу
SIGHUB - перечитать конфиги
SIGKILL - убить процесс
SIGTERM - мягко выключить программу



```bash

kill -9 (PID) - убивает процесс по PID
kill -15 (PID) - аккуратно заканчивает процесс
sudo killall  name_of_process - убивает все процессы содержащие данное имя c помощью SIGTERM
sudo killall -u username  - убивает все процессы данного юзера
```
