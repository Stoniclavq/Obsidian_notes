### tmux

 tmux - терминальный мультиплексор, с помощью которого можно запускать команду, закрыть терминал и программа закончит свою работу.
```bash
tmux new-session -s name_of_session   # запускаем сессию

Cntrl+B \"    # для запуска еще одной сессии в этом же окне  по вертикали (только двойная кавычка, без обратного слеша)

Ctrl+%        # для запуска еще одной сессии в этом же окне  по горизонтали

Ctrl+B d      # отсоединиться от сессии(сессия останется работать)

tmux attach -t name_of_session  # вернуться в существующую сессию

Ctrl+B <стрелка вверх/вниз>   # переключается между верхней и нижней сессией

Ctrl+D  #завершить сессию
```
