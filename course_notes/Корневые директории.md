boot - хранится загрузчик операционной системы GNU/LINUX
dev - хранятся файлы устройств 
etc - хранит конфигурационные файлы программ
home - хранит домашние директории всех пользователей, у которых есть доступ в linux
lib - хранит различные библиотеки, которые необходимы для работы linux
media и mnt - хранят в себе монтированные дополнительные файловые системы(флэшки, внешний диск и.т.д.)
proc - хранит в себе файлы процессов
root - домашняя директория суперпользователя root
sbin - хранит в себе исполняемые файлы суперпользователя
tmp - хранит в себе временные файлы
usr - хранит в себе пользовательские программы
var - хранит в себе часто изменяющиеся файлыx
