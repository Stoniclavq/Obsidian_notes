Файлы, в которые мы можем записать длинные команды и присвоить команде кодовое слово, при вызове которого будем исполняться данная команда. Для начала надо создать Make файл с названием Makefile, после чего записать туда нужную нам команду следующим образом:
```bash
hello:
	echo hello world
```
Во избежание конфликтов ключевых слов в make файле и названия файлов используется PHONY. То есть, если у нас есть ключевое слово hello в make файле и другой файл под названием hello, при вызове команды `make hello`она не поймет, что конкретно мы имеем в виду, файл или слово. Для разрешения этого вопросы и используется PHONY.  В make файле мы пишем следующим образом:
```bash
.PHONY: hello
hello:
	echo hello world
```
Теперь при вызове ``` make hello``` будет выполняться команда из Make файла. Также в Make файле можно писать комментарии с помощью #.

В make файлах можно при использовании одной команды предварительно запустить другие, например:
```bash
hello:
	echo "hello"
bye:hello   # здесь при запуске bye вначале запустится команда hello, а после bye
	echo "bye"
```
