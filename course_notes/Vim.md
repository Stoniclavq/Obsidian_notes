:q - выход из vim
:q! - выход из vim без сохранения файла
:w - записать в файл написанное в vim
:wq - выход из vim c сохранением в файл
Два режима:
* командный
* текстовый
#### В командном режиме:
* i  - слева от курсора
* a - справа от курсора
* A - в конец строки
* o - создает строку снизу от курсора и начинает ввод там
* gg - переходит на первую строчку файл
* G - переходит на последнюю строчку файла
* w - перейти на начало следующего слова
* W - перейти на следующую строчку
* b - перейти на предыдущее слово
* B - перейти на предыдущую строчку
* e - перейти на конец следующего слова
* (число)(команда) - пишем сначала число, после этого пишем команду и она выполняется ровно такое число раз
* f (символ) -  переходит к заданному символу текущей строки
* /(символ или набор символов) - поиск символа или набора символов по всему файлу снизу от курсора (нажать n для перехода к следующему найденному символу)
* ?(символ или набор символов) - поиск символа или набора символов по всему файлу сверху от курсора (нажать n для перехода к следующему найденному символу)
* m(название заметки) - создание заметки на строчку с названием из одного символа
* '(название заметки) - перемещение к заметке с данным заранее заданным названием
* h - влево
* j - вниз
* k - вверх
* l - вправо
* v - войти в визуальный режим для выделения текста 
* V - выделить всю строку
* x или d - удалить выделенный текст и сохранить в буфер vim
* p - вставить текст из буфера справа от курсора
* P - вставить текст из буфера слева от курсора
* dw - удалить слово
* u - вернуть удаленное слово
* dd - удалить строчку
* cw - заменить слово на другое слово
* С - заменить текст до конца строки
* D - удалить  строку справа от курсора
* q(номер макроса) - начать запись макроса 
* @(номер макроса) - исполнить макрос
