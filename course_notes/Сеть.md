Концентратор - прибор, в который проводится много сетевых кабелей из компьютеров и служит, как хаб. При попытке отправить данные с одного компьютера на другой, которые подключены к концентратору, сообщение отправится на все подключенные компьютеры.

Коммутатор (switch) - Очень схожая вещь с концентратором, но так как он запоминает MAC-адреса всех подключенных устройств, поэтому при отправки сообщения с одного на другое устройство, то оно приходит именно туда. Это намного безопаснее и удобнее.

Маршрутизатор(роутер) -  работает с IP-адресами передает трафик из одной сети в другую сеть

INTRONET - локальная сеть, состаящая из множества коммутаторов

MAC-адрес - уникальный адрес сетевой карты

InterNet - Объединение большого кол-ва  сетей

Белый IP-адрес - статичный IP-адрес, получается через провайдер

### Прикладной уровень
- протокол FTP расшифровывается как File Transfer Protocol, то есть протокол передачи файлов, его можно использовать для загрузки файлов на сервер;
	
- протокол SMB, который используется Windows в частности для работы с сетевыми директориями с общим доступом, о которых мы говорили выше;
    
- протокол HTTP (HyperText Transfer Protocol, протокол передачи гипертекста) используется всеми веб-браузерами для загрузки контента веб-страниц;
    
- HTTPs это HTTP с шифрованием трафика (Secure HTTP, безопасный HTTP);
    
- протокол SSH (Secure Shell, безопасная оболочка) позволяет удалённо подключаться к компьютерам и управлять ими;
    
- DNS протокол (Domain Name System, система доменных имён) служит для получения информации о доменах (например, о домене `google.com`).

DHTP - Dynamic Host Configuration Protocol (Автоматическое получение устройств IP-адресов)
### Протоколы TCP и UDP(транспортные)

TCP - протокол, который гарантирует доставку пакетов данных, но у него невысокая скорость передачи
UDP - протокол, который передает данные намного быстрее TCP, но не гарантирует их целостную доставку
