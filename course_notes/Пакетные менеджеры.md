### apt 

```bash
sudo apt install programm_name  # Установка программы

sudo apt remove programm_name  # Удаление программы

sudo apt purge programm_name  # Удаление программы вместе с конфигами

sudo apt update  # Обновление информацию о пакетах

sudo apt upgrade # Обновление всех пакетов

apt search package_name  # поиск пакетов

apt show package_name # информация о пакете
```
### brew (MacOs)

```bash 
brew install programm_name  # Установка программы
```
