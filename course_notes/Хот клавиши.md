## В консоли Unix подобных
Ctrl + D  - для выхода из интерпретатора 
Ctrl + A - для перемещение в начало строки
Ctrl + E - для перемещение в конец строки
Ctrl + W - для удаление слова слева от курсора
Ctrl + Y - для возврата последнего удаленного элемента с помощью ctrl+w
Ctrl + (стрелки влево и вправо) - перемещение по словам 
Ctrl + U - удаление всего, что слева от курсора
## Другое
Ctrl + Q (в IDLE) - выход из IDLE
