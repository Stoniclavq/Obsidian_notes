[[Linux]]

Сохраняются только на 1 сеанс, если хотим, чтобы осталось на последующие сеансы, записываем в bashrc

Чтобы добавить данные к переменной окружения, у которой уже есть какие-то данные, можно в файл bashrc написать следующее:
```bash
export PATH = $PATH:path_to_directory       # пример конкретно под переменную PATH
```

EDITOR - отвечает за дефолтный редактор
USER - Хранит в себе имя текущего пользователя
HOME - хранит в себе путь к домашней директории текущего пользователя
PWD - хранит в себе путь от корневой директории до текущей директории
PATH - хранит в себе все директории, в которых система будет искать исполнимые файлы
CDPATH - хранит в себе директории, в которых команда cd будет искать директории
