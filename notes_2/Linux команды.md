## Базовые команды

*  pwd - показывает директорию, в которой сейчас находишься до корневой директории
```bash
pwd
```
*  cd - перейти в указанную директорию
```bash
cd path_to_directory

cd -    ## переместиться в предыдущее директорию, в которой мы были
```
*  ls - вывод содержимого директории
```bash
ls path_to_dir
```
ls -a - вывод скрытых файлов
```bash
ls -a
```
ls -l  - вывод данных списком
```bash
ls -l
```
ls -R - выводит содержимое директорий и поддиректорий данной директории
```bash
ls -R path_to_directory
```
* cat - вывод содержимого файла в консоль
```bash
cat path_to_file

		Ctrl + D   # выход из команды cat
```
* echo - вывод данных из консоли в консоль или в другой файл
```bash
echo some_data > path_to_file
```
* rm - удаление директории
```bash
rm path_to_directory


rm -r path_to_directory    ## удаляет непустые директории
```
*  touch - создание пустого файла
```bash
touch name_of_filehistor

touch -m --date="date" path_to_file   # Изменение времени Modify у файла

touch -a --date="date" path_to_file   # Изменение времени Access у файла

touch -t  202508141500 path_to_file   # Изменение времени Access и Modify у файла
```
* mkdir - создание директории
```bash
mkdir name_of_directory


mkdir -p 1/2/3   ## позволяет создавать директорию и поддерикторию одновременно
```
* | - передает вывод команды слева от черты на вход в команду справа от нее
```bash
first_command | second_command
```
   * history - выводит историю всех команд, написанных в консоли 
   ```bash
history 

history | less  ## выводит данные истории постранично

```
*  grep - ищет в тексте заданные данные и выводит строчки с найденными данными
```bash
grep 'some_text' path_to_file

-r  # производить поиск рекурсивно
-i  # поиск не зависимо от регистра
-n  # результат выводить с номерами строк
--include="something"  # поиск по названию файла
```

* ripgrep - удобный и улучшенный аналог grep
```bash
rg "something"
```
*  cp - копирует данные одного файла в другой файл
```bash
cp path_to_first_file path_to_second_file
```
* man - выводит справку о заданной команде
```bash
man name_of_the_command
```
* sort - сортировка по 1 числу
```bash
sort


sort -h  ##сортировка по числам
```
* printf - вывод с учетом форматирования 
```bash
printf "text\ntext"
```
* tac - вывод файла с конца
```bash
tac path_to_file
```
* wc -  считает кол-во слов, строк и байт в файле
```bash
wc path_to_file


wc -l path_to_file   # кол-во строк в файле
wc -c path_to_file   # кол-во байт в файле
wc -w path_to_file   # кол-во слов в файле
```
*  head - вывести первые строчки файла

```bash
head -n1 file_name   # выводит 1 строчку сверху файла
```
* tail  - вывести последние  строчки файла

```bash
tail -n3 file_name   # выводит 3 строчку снизу файла
```
* cut - выводит  заданное число символов в каждой строке
```bash
cut -c 1-3 file_name   # выводит 1-3 элемент каждой строки
```
* sudo - права суперпользователя
* mv -  переместить файл в другое место файловой системы(также можно его переименовать)
```bash
mv path_to_file new_path_to_file
```
*  env - выводит все переменные окружения
```bash
env
```
* ln - создание жесткой ссылки
```bash
ln path_to file path_to_hard_link


ln -s path_to file path_to_soft_link   # создание мягкой ссылки
```
* chgrp - меняет группу в правах доступа
```bash
sudo chgrp new_group file_name
```
* chmod - изменения прав доступа к файлу
```bash
chmod 777 file_name
chmod u+x file_name
chmod o-r file_name
chmod +x file_name
```
* chown - меняет владельца файла
```bash
sudo chown username file_name
```
* find - поиск по названиям файлов и их атрибутам
```bash
find или find . или find . -print   # показывает все содержимое текущей директории списком

find . -name "something" # поиск по названиям файлов по маске

find . -maxdepth 1 -name "something" # поиск по названиям файлов по маске только в текущей директории(можно искать и во вложенных, изменив цифру 1)

find . -type f  # поисе идет только по файлам
find . -type d  # поиск идет только по директориям
find . -type d -path path_to_directory -prune -o -name "something" -print   # исчем все файлы по маске, кроме определенной директории(-path указавыть путь до файла/директории, -prune - пропустить полученные данные в поиске, -o - логическое или)

find . -iname "something" # поиск по названиям файлов по маске не учитывая регистр


find -atime 0 # ищем файлы/директории по Access time,то есть которые открывались за последние сутки(ровно 24 часа от этого момента). Если изменить 0 на 1, то будут искаться те файлы, в которые мы последний раз заходили от 24 до 48 часов от текущего момента. И так далее по дням.

find -mtime 0  # ищем файлы/директории по Modify time


find -anewer file_name  # ищем все файлы, в которые заходили позже, чем в заданный файл(Access time)
find -cnewer file_name  # ищем все файлы, в которые происходили изменения  позже, чем в заданном файле(Change time)

find -size 100  # поиск файлов по заданному размеру (+!00 - больше 100, -100 - меньше 100)

find -size +100 -a -size 150  # -a - логическое и

find -exec command {} \;  # выполеяет команды для результатов поиска

find -delete   # удалить результаты поиска
```
* split - разделить файл на несколько файлов
``` bash
split -b 100M path_to_file "new_files_name" # файл разобьется на блоки размером 100M 
```
* fallocate - создать файл с определенным размером
```bash
fallocate -l 10M file_name # создается файл размером 10M
```
* md5sum - хэш-функция
```bash
md5sum file_name # на выход получаем хэш файла
```
* cmp - сравнение файлов
```bash
cmp file_name1 file_name2   # если файлы одинаковые, то ничего не выводит
```
* ps - показывает все процессы
```bash
ps
```
* jobs - показывает все свернутые процессы(Ctrl+Z, чтобы свернуть любой процесс)
```bash
jobs
```
* fg - чтобы отрыть свернутый процесс в консоль
```bash
fg

fg %1       # открывет процесс с индексом 1
```
*  bg - продолжает свернутый процесс, но делает это в background, то есть процесс не выводится в консоли и в ней можно продолжать писать команды, пока процесс активен.
```bash
bg

bg %1       # открывет процесс с индексом 1
```
## Опциональные команды

* xxd - вывод содержимого файла в байтах (так называемый hex-редактор )
```bash
xxd path_to_file
```
* ip a - выводит в консоль нынешний ip-адрес 
```bash
ip a
```
* * tree - вывод содержимого директории в виде дерева
```bash
tree 
```
* date - показывает текущую дату и время
```bash
date
```
* !номер строчки - выполняется команда с этой строчки взятая из истории команд в консоли
```bash
!number     ## пример !22 - выводит 22 строчку из истории

!!    ## исполняет предыдущию команду

!command  ## исполняют последнюю написанную данную команду
```
* du -sh - выводит размер файлов/директорий
```bash
du -sh path_to_file
```
* tee - "тройничок" или "вилка" -  выводит результат команды в стандартный поток вывода и записывает в файл
```bash
ls -la | tee output.txt
```
* * watch - повторяет заданную команду раз  в 2 секунды
```bash
watch command

watch -n 1 command  # запускается раз в 1 секунду

watch -g "command"    # запускает программу до того момента пока ее вывод не изменится


Ctrl + C - выйти из режима 
```
*   df - показывает свободное место на дисках
```bash
df -h  # показывает данные удобно для глаз человека
```
* stat - показывает статистику файла
```bash
stat path_to_file
```
* whoami - выводит имя текущего пользователя
```bash
whoami
```
* type - показывает информацию  о типе команды
```bash
type command
```
*  which - показывает путь к данному файлу в системе
```bash
which command
```
* tr - переводит заданную последовательность символов в другую заданную последовательность символов
```bash
tr 'symbols' 'another_symbols'
```
* groups - показывает все группы пользователя, в которых он состоит
```bash
groups username
```
* adduser - добавляет нового пользователя (исполняется с sudo)
```bash
sudo adduser username
```
* su - заходит из текущего пользователя в другого, которого мы укажем
```bash
su - username
```
* groupadd - добавление новой группы пользователей
```bash
sudo groupadd groupname
```
*  usermod - добавление новых пользователей в группу
```bash
sudo usermod -aG group_name username
```
* passwd - изменить пароль пользователю
```bash
sudo passwd username
```
* deluser - удаляет пользователя из группы
```bash
sudo deluser username group_name
```
* tar - архивация файла 
```bash
tar -cvf archieve_file file  # Только архивирует файлы в один без сжатия

tar -czvf archieve_file.gz file   # Сжатия файла

tar -xzvf archieve_file.gz file # Распаковка сжатого файла
```
* zip - архивация файла
```bash
sudo apt install -y zip
sudo apt install -y unzip

zip -r archieve_file file # сжатие и архивация

unzip arcieve_file # распаковка
```
* locate - поиск файла по имени
*  wget - скачивание программ через исходники
```bash
wget https://... 
```
*  curl - позволяет более гибко скачивать файлы из сети
* lscpu - показывает характеристики cpu
*  tmux - терминальный мультиплексор, с помощью которого можно запускать команду, закрыть терминал и программа закончит свою работу.
```bash
tmux new-session -s name_of_session   # запускаем сессию

Cntrl+A \"    # для запуска еще одной сессии в этом же окне (только двойная кавычка, без обратного слеша)

Ctrl+A d      # отсоединиться от сессии(сессия останется работать)

tmux attach -t name_of_session  # вернуться в существующую сессию

Ctrl+A <стрелка вверх/вниз>   # переключается между верхней и нижней сессией

Ctrl+D  #завершить сессию

Ctrl+A ,  # переименовать текущую сессию

Ctrl+A r   # перезагрузить текущую сессию

Ctrl+A e  # открывает конфиг tmux

Ctrl+A Enter  # вход в режим копирования
```

* pstree - выводит дерево всех процессов (нужно установить psmisc)
```bash
pstree -a 
```

* htop - показывает все процессы и позволяет управлять ими
```bash
htop
```
* ffmpeg - позволяет обрабатывать видео, аудио и изображения
* bat - аналог cat, но с подстветкой синтаксиса содержимого
